This is LedScreen Qt5 QML component.

Included are three different examples (hello, billboard, hello) demonstrating 
the usage of the component. Use Qt5 'qmlscene' binary to run these examples. 
To use the component in your own application just copy the "ledscreencomponent" 
directory, import it and use "LedScreen { ... }" in your QML files.
