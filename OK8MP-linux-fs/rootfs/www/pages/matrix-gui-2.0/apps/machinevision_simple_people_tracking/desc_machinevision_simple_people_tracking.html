Simple People Tracking TOF Demo

In order to run this demo you need to have OPT8241-CDK-EVM attached to USB2.0 port.

More information about OPT8241-CDK-EVM can be found at http://www.ti.com/tool/opt8241-cdk-evm
The People Counting demo using 3D ToF reference design is a subsystem based on TI ToF
image sensor combined with tracking and detection algorithms to count the number of
occupants present in a given area. Complete description of the demo can be found
at http://www.ti.com/lit/pdf/tidud06
