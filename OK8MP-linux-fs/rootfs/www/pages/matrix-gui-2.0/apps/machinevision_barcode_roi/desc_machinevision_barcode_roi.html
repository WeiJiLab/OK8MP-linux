Detecting Barcode ROI Example

This example demonstrates barcode detection using computuer vision and image processing techniques.

The major steps of this demo include:
    1. Read the input barcode image (/usr/share/ti/image/sample_barcode.jpg)
    2. Detects the barcode area, and mark it with a green bounding box
    3. Display the barcode image with the green rounding box
    4. The display is terminated after ~20 seconds
